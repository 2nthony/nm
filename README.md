# sao-evillt-nm

# Usage

`sao-evillt-nm` is a generator for [SAO](https://github.com/saojs/sao), install is first:

```sh
yarn global add sao
# Prefer npm
npm i -g sao

#####################

# From npm
sao evillt-nm my-project
# From git
sao evillt/sao-evillt-nm my-project
```
